HTML TAGS

HTML is completely enclosed in tags

What is Tag?

Text palced between '<' and '>' is called as tags


Tags are classified into two types
1. Paired tags
2. Non-Paired tags

1. Paired tags - The tags that have both opening and closing tags are called as paried tags.Example-:html tag
Note:- The closing tags start with '/'


2. Non-Paired - The tags that have only opening tag but no closing tag are called as non-paired tags. - These tags are also called as
i) Self closed tags
ii)Forcefully closed tags
iii)Empty tags
iv)Singular tags.
Eg -:Break tag(line break)
Hrizontal rule (h. line)
image tag
