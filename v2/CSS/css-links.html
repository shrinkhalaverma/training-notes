CSS Links

Links can be styled with color, font-family, background, etc.
links have following states

a: link a normal visited link
a: visited visited by user
a: hover mouse pointer on link
a: active moment it is clicked
Example-:
      a:link
     {
      color : pink;
     }
      a:visited
     
      color : green;
     }
      a:hover
     {
      color : red;
     }
      a:active
     {
      color : blue;
     }
