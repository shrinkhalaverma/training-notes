CSS Borders

1. border-width specifies width of border
border-width : number
     : thin
     : medium
     : thick
Example-:
     .thk
     {
      border-width: 15px;
     }

2. border-color: any color
Example-:      .brdr1
      {
      border-style: ridge;
      border-width: 10px;
      border-color: red;
     }

3. border-sides : can apply specific styles to specific border
Example-:
     .brdr4
     {
      border : 15px orange solid;
      border-left: 10px blue ridge;
      border-bottom: 10px navy inset;
      border-right: skyblue 10px double;
     }

4. border-images

border-image-source source of image
border-image-slice used to slice image
border-image-width used to set width of image
border-image-repeat round,repeat,stretch

5. CSS Outline-  Same as that of border. Outline comes after border. Sides, radious, image, 3d styles are not supported

6. CSS Margin -   Outer Space

margin-top number / %
margin-bottom number / %
margin-right number / %
margin-left number / %
margin-auto Horizontally center of container

7. CSS Padding-:  Inner Space

padding-top number / %
padding-bottom number / %
padding-right number / %
padding-left number / %
padding number ->  all sides
