CSS Backgrounds

1.  Background-color specifies background color of element
Example-:
   <h1 style="background-color: rgb(0,0,255);"> I am heading 1</h1>
   <div style="background-color: rgba(0,255,255,0.5);">I am division</div>
   <p style="background-color: #ff0000;">I am pragraph</p>
   <span style="background-color:hsla(100, 50%, 50%, 0.5);">I am span</span>

2.  Background-image adds image to background
background-image :   url();

3.  Background-repeat repeats images either in horizontal direction or in vertical direction or no repeat

4.  Background-position
     right / left
     top / bottom
Example-:
     background-position : right top;

5.  Background-attachment: specifies image should be fixed or scrolling
i)  background-attachment:  fixed ;(not fixed with the page)
ii)  background-attachment:  scroll;(fixed with page)

6.  Background-size specifies size of background
Example-:
    background-size : 100px 15px;

7.CSS Gradient backgrounds
1.  Linear gradient
2.  Radial gradient
Note :- minimum 2 color stops required 1. Linear gradient
background : linear-gradient(direction, color_stop1, color_stop2, color_stop3, ...)
direction -> to right
        -> to left
        -> to bottom
        -> to top
        -> to right bottom
        -> ...
Example-:
     background: linear-gradient(to right, red,black);
