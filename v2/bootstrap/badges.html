Badges/Breadcrumbs/Spinner

1. Badges can be used as part of links or buttons to provide a counter.
Pill-Badge- Use the .badge-pill modifier class to make badges more rounded (with a larger border-radius and additional horizontal padding).

2. Breadcrumbs - breadcrumb "/" character or separator
Classes-:
     breadcrumb
     breadcrumb-item

3. Spinner-:  Bootstrap “spinners” can be used to show the loading state in projects. They’re built only with HTML and CSS.
classes-:
     spinner-border
     spinner-grow
     spinner-border-sm
     spinner-grow-sm

Types of CSS

1.inline CSS Highest Priority can be applied to only that tag.
2.Internal CSS Medium priority can be applied to only that document
3.External CSS Lowest priority can be applied globally
