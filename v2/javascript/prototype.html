Prototype

1. Prototype-:  Prototype is a property which adds members to constructor dynamically.
Example-:
    function class_one(){}
    class_one.prototype.wish = `Welcome to Prototype`
    console.log(new class_one().wish)

2. Prototype Chaining -:
    Nesting one constructor prototype to another constructor is called as prototype chaining.
    Concept of inheritance is achieved with prototype chaining.
Example-:
    class_one.prototype.sub_one = "Angular"
    class_one.prototype.sub_two = "NodeJS"
    class_one.prototype.sub_three = "MongoDB"
    let obj = new class_one()
    console.log(obj.sub_one, obj.sub_two, obj.sub_three)

3. Passing parameters to functions-:
Default parameters
It allows named parameters to be initialized with default values if no value or undefined is passed.
Example-:
    function fun_one(arg1 = 10, arg2 = "FullStack", arg3 = "Angular")
    {
    console.log(arg1, arg2, arg3)
   }

4. Optional Parameters-:
While calling a function no need to pass all arguments. We can keep few arguments as optional. Optional parameters introduced in ES6.
Example-:
    function fun_one(arg1, arg2, arg3)
   {
    console.log(arg1, arg2, arg3)
   }

5. Rest Parameters-:
it is improved way to handle function parameters. It allows us to represent infinite number of arguments as an array. rest parameters represented by '...'
Example-:
    function fun_one(...arg)
   {
    console.log(arg)
   }
