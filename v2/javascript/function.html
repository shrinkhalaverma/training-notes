Functions

1. Functions-:
A particular business logic is called as function.
Functions are used to reuse business logic.
We have following type of functions
     i) Named Functions
     ii) Anonymous Functions
     iii) Constructor Function

We can pass various parameters to functions.
     i) Rest Parameters      ii)Optional Parameters      iii)Default Parameters

3. Numbers-:
    Javascript supports decimal, double, hexadecimal, octal and binary numbers.
Example-:
    var decimal = 100
    console.log(decimal)
    var double = 12.43
    console.log(double)

1. Named Function

Function with name is called as named functions
Syntax
   //function definiton
    function function_name(parameters)
   {
   //business logic
   }
   //calling function
    function_name(arguments)

2. Anonymous Function

The function without name is called as anonymous function. these are also called as arrow functions or callback functions. we can represent arrow functions by '=>' arrow symbol. Arrow functions introduced in ES6. Arrow functions are more secured than named functions. anonymous functions utilize heap memory efficiently.
Syntax-:
   //function definition
    var / let variablename = (arguments) => {
   //business logic
    }
   //calling function
    variablename(parameters)

3. Constructor Function

Constructor Function is used to create class like structure. In constructor function everything (variables and functions) should start with 'this' keyword. We can create object to constructor using 'new' keyword.
Example-:
    function class_one()
   {
    this.wish = "Welcome to constructor functions"
   }
    var obj = new class_one()
    console.log(obj.wish)
