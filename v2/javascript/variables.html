Variables

1. Variables-:
     Variables are used to store data. by using variables we can      store any type of data, Eg numbers, strings, files, audio,      video, etc. Variables can be declared using 'var', 'let',      'const'. 'let' and 'const' keywords introduced in ES5.

2. String
Strings can be declared with '', "", ``
- `` is introduced in ES6.
- `` is used to define multiline strings.
Example-:
    var wish = 'Good afternoon'     console.log(wish)

3. Numbers-:
    Javascript supports decimal, double, hexadecimal, octal and binary numbers.
Example-:
    var decimal = 100
    console.log(decimal)
    var double = 12.43
    console.log(double)

4. Boolean-:A JavaScript Boolean represents one of two values: true or false.
Example-:
    var flag = true
    console.log(flag)
    var flag = false
    console.log(flag)
