HTML Tags

1.HTML Heading

In html there are six heading tags. All heading tags are paired tags.
<h1>Heading 1</h1>
<h2>Heading 2</>

Attributes Parameters
align Left ,Center,Right

2.HTML Paragraph

It is used to divide text into different paragraphs it is a paired tag
<p>I am first paragraph</p>
<p>I am second paragraph</P>

Attributes Parameters
align Left ,Center,Right,Justify

3.HTML line break(<br/> tag)

br stands for breaks it is used to break a line and shift following text to next line. it is a non paired tag.

Non breaking space --> &nbsp; it is used to insert more than one spaces. it is special character entity
HTML Horizontal rule(<hr/> tag)- This tag is used to draw a line across the web page. it is a non paired tag

4.HTML forms(<form> tag)

This tag defines a from that is used to collect user inputs. It is a paired tag.
Form Attributes-:   i)method  ii)action  iii)target  iv)encrypt

5. <meta/> tag

Metadata is data about data. <meta/> tag provides metadata about html document. Metadata is not displayed on web page but it can be machine parsable (understandable).
meta attributes-:    i)meta Keywords  ii)meta description  iii) meta author  iv)meta charset  v)meta viewpoint

6.<style> tag

It is used to apply styles to html documents. it is a paired tag
Example-:
<!doctype html>
<html>
<head>
      <title>Styling Example</title>
      <style>
         p{
         color:red;
         font-family : roboto,candara;
         font-size : 50px;
         }
      </style>
</head>
<body></body>
</html>

7. <script> tag

It is paired tag. it is used to apply script to html page
Example-:
<!doctype html>
<html>
<head>
<title>Script demo</title>
<script type = "text/javascript" language = "javascript">
function myAlert()
{
alert("Button clicked")
}
</script>
</head>
<body>
<h1></h1>
<p></p>
<button onclick = "myAlert()"> Click Me </button>
</body>
</html>
