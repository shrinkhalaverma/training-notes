Lists in HTML

Lists

There are three types of lists in html.
1. Ordered List                <ol> <li>
2.Unordered list    <ul> <li>
3.Definition list    <dl> <dt> <dd>

Ordered List(<ol>)

It is also called as numbered list. It is used to give numbering to list items.

Attributes Parameters
Type i, I, a, A, 1
Start Any number

Unordered List(<ul> )

Attributes Parameters
Type disc,circle,square

Definition List

The <dl> tag defines a Definition list. The <dl> tag is used in conjunction with <dt> (defines terms/names) and <dd> (describes each term/name).
