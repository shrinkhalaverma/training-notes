HTML Section

Sections in html

There are two sections in html.
1.Head Section
2.Body Section

Head Section

This section contains root element as <head> tag
- It is a a pired tag
- This tag contain following tags

1. <title>    -Document title
2.<link/>    -Link to external document
3.<meta/>  -Meta information
4.<style>    -Used for styling
5.<script>   -Used to provide script

Body Section

- it contain <body> tag as root element.
- this section contains contents of html document, such as text, hyperlinks, images, special characters, lists, forms, etc.
- <body>is paired tag
