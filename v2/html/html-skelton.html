Working with HTML

1.Open any text editor for eg notepad, notepad++, sublime text, etc Save this file with any name and extension '.html' or '.html' Eg 'index.html'
2.Run above file in any web browser eg. MS Internet explorer, MS edge, google chrome, MFF, Safari, Opera, UCB, etc
Example-:
   <html>
      <head>
         <title>My first web page </title>
      </head>
<body>
Hello world
         <h1>Welcome to Fist HTML session </h1>
</body>
</html>

Note :- HTML is case insensitive means A and a are same
