CSS BORDERS

border-width specifies width of border .
border-width number :thin
                                   :medium
                                   :thick
Syntax:
.thk
{
border-width: 15px;
}
div
{
width : 200px; height : 100px;
}

border-color: any color
Eg:
.brdr2
{
border : maroon 2px solid;
}

CSS Outline

Same as that of border
Outline comes after border
Sides, radious, image, 3d styles are not supported
                                Eg:
                                .olbr1
                                {
                                border : 10px solid black;
                                }

CSS Margin

CSS Margine : -> Outer space
                margine-top : number / %
                margine-bottom : number / %
                margine-right : number / %
                margine-left : number / %
                margine : auto -> Horizontally center of container
