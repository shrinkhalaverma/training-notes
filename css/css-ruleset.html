CSS SYNTAX

CSS syntax (rule-set)

CSS syntax is selector and declaration block. The selector points to the html element you want to style.
Each declaration includes CSS property and value pair.
Propertes and values are separated by colon(:) Each property and value separated by semicolon(;)
Eg
h1
{
color : red;
font-size : 12px;
}
where, h1 is a selector color is property red is value font-size is property 12px is value
