CSS PADDING

CSS Padding : -> Inner space
padding-top : number / %
padding-bottom : number / %
padding-right : number / %
padding-left : number / %
padding : number -> all sides
        Syntax-:
            .pdng1
            {
            padding-top: 50px;
            padding-bottom: 50px;
            padding-left: 50px;
            padding : 100px;
            }
